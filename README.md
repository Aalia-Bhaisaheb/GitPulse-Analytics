# ⚡ GitPulse Analytics

> Turn your GitHub stats into AI-powered insights (or insults).

GitPulse Analytics is a full-stack application that fetches real-time data from the GitHub API and uses the **Google Gemini Pro AI** to analyze a user's coding habits. Whether you need a ego-boost (Toast) or a reality check (Roast), GitPulse delivers.

## ✨ Features

* **AI Roast & Toast:** Get personalized feedback on your repositories using Google Gemini AI.
* **Bento Grid UI:** A modern, responsive dashboard layout built with Tailwind CSS.
* **Real-time Stats:** Fetches your latest repository counts, followers, and public data directly from GitHub.
* **Monorepo Architecture:** Clean separation between the React frontend and Node.js backend.

## 🛠️ Tech Stack

* **Frontend:** React, Vite, Tailwind CSS, Lucide Icons (Hosted on **Vercel**)
* **Backend:** Node.js, Express (Hosted on **Render**)
* **AI Integration:** Google Gemini Pro API
* **Source Data:** GitHub REST API
